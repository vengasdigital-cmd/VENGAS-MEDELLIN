# VENGAS Medellín

Esta carpeta ahora tiene `index.html` para que GitHub Pages muestre la web principal.

## Cómo publicar
1. Confirma los cambios con `git add index.html README.md` y `git commit -m "Publicar página principal en GitHub Pages"`.
2. Sube al repositorio remoto con `git push origin master`.
3. En GitHub, ve a Settings > Pages y elige la rama `master` / carpeta `/ (root)`.
4. Accede desde `https://vengasdigital-cmd.github.io/VENGAS-MEDELLIN/`.
